VR Escape Room

Players will use the teleport pads to travel around solving puzzles in the Escape Room. There are three puzzles in this room:

The first is when the user places the skull on the button on the table. The door will unlock if the skull is confirmed to have collided with the button.

The second is when the user clicks on the candles in order. The door unlocks if it is confirmed that user has clicked on each candle in order. If the wrong candle has been clicked, the candle will reset positions.

The third is when the user places the right object on top of the rock. If the wrong object is placed on top, the door will not unlock. It will only unlock when it detects that the right object has collided with the rock.

Android Studio

Simple To Do App

Using java to create classes, arrays and adapters to create a list application. I created a spinner with the options to add or remove an item.
When "Add a new task" is selected, the delete button is disabled and the add button is enabled. User can enter an item and add into the list.
When "Remove a task" is selected, the add button is disabled and the delete button is enabled. User can then enter a number (index) to remove an item.
I used switch to set which button should be enabled, and onClick to add or remove item. When an item is added or removed, it notifies the array that it has a change, which then displays the new results.

Application used: Android Studio
